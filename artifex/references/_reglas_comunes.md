# Reglas Comunes para Todos los Agentes

> **ATENCIÓN:** Las reglas de estilo se han consolidado en
> `_estilo_salida.md`. Ese archivo es la única fuente de verdad para
> todo el formato de salida (LaTeX, colores, headings, ICFES,
> prohibiciones, template completo, etc.). **Leer `_estilo_salida.md`**
> antes de generar cualquier contenido.

## Casos borde comunes

| Situacion | Accion |
|:---|:---|
| Entrada insuficiente o incompleta | Detener ejecucion y reportar al orquestador |
| Error en paso anterior | No inventar datos faltantes; reportar |
| Formato incompatible | Usar Markdown estandar; evitar formatos propietarios |
